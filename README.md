# PoC NCKU phishing

# Usage

* install node and npm
```
sudo apt get install nodejs
sudo apt get install npm
```
* install node modules
```
npm install
```

* run

	* option 1

	terminal 1
	```
	node server.js
	```

	terminal 2
	```
	sh setup_ngrok.sh
	```

	* option 2
	```
	sudo python3 start.py
	```
